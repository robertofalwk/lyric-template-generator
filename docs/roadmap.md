# Product Roadmap - Lyric Lab

## Phase 1: Stability & UX (Current)
- [x] Professional Domain-Driven Architecture
- [x] Zod validation for all data models
- [x] Job Queue for background rendering
- [x] High-fidelity template system (8 initial styles)
- [x] Premium Dashboard UI

## Phase 2: Intelligence & Media
- [ ] **Multi-track support:** Add background video/image overlays.
- [ ] **AI Vocal Isolation Improvement:** Integration with deeper Demucs models.
- [ ] **Smart Transcription:** Auto-generate lyrics from audio if not provided.
- [ ] **Dynamic Backgrounds:** Generative AI art based on lyrics.

## Phase 3: Scaling & Platform
- [ ] **S3 Integration:** Move from local storage to cloud.
- [ ] **Postgres/Supabase:** Move from JSON files to a real DB.
- [ ] **Distributed Workers:** Use BullMQ + Redis to scale render workers.
- [ ] **User Auth & Credits:** SaaS billing and project ownership.

## Phase 4: Creative Tools
- [ ] **Advanced Animation Editor:** Keyframe-based control for power users.
- [ ] **Lottie Integration:** Use vector animations in templates.
- [ ] **Multi-language OCR:** Upload lyrics from images/screenshots.
